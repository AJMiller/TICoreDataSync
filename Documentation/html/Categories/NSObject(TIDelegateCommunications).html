NSObject(TIDelegateCommunications) Category Reference

Declared in NSObject+TIDelegateCommunications.h

Overview

TIDelegateCommunications is a category on NSObject to make it easy to communicate with a delegate.

The arguments passed to any of these methods must be objects.
